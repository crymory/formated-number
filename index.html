Python implementation:

def format_phone_numbers(numbers_text):
    # Разделяем текст на отдельные номера (по пробелам или новым строкам)
    raw_numbers = numbers_text.split()
    # Обрабатываем каждый номер - убираем префикс "38"
    formatted_numbers = []
    for number in raw_numbers:
        if number.startswith("38"):
            formatted_number = number[2:]  # Убираем первые 2 символа (38)
            formatted_numbers.append(formatted_number)
        else:
            # Если номер не начинается с 38, оставляем как есть
            formatted_numbers.append(number)
    # Возвращаем отформатированные номера в столбик
    return "\n".join(formatted_numbers)

JavaScript implementation (used on this site):

function formatPhoneNumbers(numbersText) {
    // Split the text into individual numbers
    const rawNumbers = numbersText.split(/[\s\n]+/);
    // Process each number - remove "38" prefix
    const formattedNumbers = rawNumbers.map(number => {
        if (number.startsWith("38")) {
            return number.substring(2); // Remove first 2 characters (38)
        } else {
            return number; // Keep as is if doesn't start with 38
        }
    });
    // Return formatted numbers as a column
    return formattedNumbers.join("\n");
}

About This Tool

This simple phone formatter tool was created to help format Ukrainian phone numbers by removing the "38" country code prefix. This is useful when you need to quickly format multiple phone numbers for local use.

Features

  • Removes "38" prefix from phone numbers
  • Processes multiple numbers at once
  • Works with numbers separated by spaces or new lines
  • Preserves numbers that don't have the "38" prefix

Usage

Simply paste your phone numbers in the input field, click "Format Numbers", and copy the formatted result.
